# Analisi qualità dell'aria - Milano (Esempio)


Piccolo progetto di esempio con dati di qualità dell'aria (dataset di prova). Contiene:


- `data/sample_air_data.csv` — dataset di esempio (CSV)
- `notebooks/analysis.ipynb` — notebook per esplorazione e visualizzazioni


## Come usare


1. Copia i file nelle rispettive cartelle del tuo repository (rispettare i percorsi `data/` e `notebooks/`).
2. Crea un virtual environment e installa le dipendenze:


```bash
python -m venv .venv
source .venv/bin/activate # su Windows: .venv\Scripts\activate
pip install pandas matplotlib jupyter
